# BFGSxSIMD
Improvements and SIMD implementation for the LBFGS++ library

#### The full changes are not yet avaiable here.

We used AADC by Matlogica (matlogica.com).
One can add it in the repository root to use our full project.

The files here avaible are the initial simple changes to LBFGS++ and do not increase efficiency for the Rosenbrock function.

The other tests done with AADC and AVX for Rosenbrock, LMM and ORE will be pushed when reviewed.
